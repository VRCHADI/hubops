# HubOps

Hub de operações/logística desenvolvido com Java, Spring Boot e arquitetura hexagonal.

## Tecnologias

- Java 21
- Spring Boot
- MySQL
- JPA/Hibernate
- Docker
- JWT
- Swagger/OpenAPI

## Arquitetura

O projeto utiliza arquitetura hexagonal, separando domínio, aplicação e infraestrutura.

## Objetivo

Simular um ambiente corporativo real com boas práticas de backend e integração frontend/API.

## Estrutura inicial

```text
src/main/java/com/gft/hubops
├── adapters
├── application
├── config
└── domain
```
